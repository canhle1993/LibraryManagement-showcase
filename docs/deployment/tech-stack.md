# Tech Stack And Local Run Notes

## Core Stack

- Java SE 21
- Java Swing
- Maven
- SQL Server

## Main Libraries

- JDBC SQL Server driver
- JasperReports
- Apache POI
- ZXing
- OpenCV / JavaCV
- FFmpeg support for camera processing
- BCrypt
- Log4j
- JSON utilities

## Development Setup

According to the project report, the project was developed and run with:

- Eclipse
- Maven project structure
- SQL Server database import
- `db.properties` configured with database credentials
- `src/main/java/gui/Main.java` used as the main entry point

## Public Showcase Note

This repository is not intended to be a runnable public distribution of the application.

Instead, it documents:

- what the system does
- how the modules fit together
- what technologies were used
- what I personally implemented
