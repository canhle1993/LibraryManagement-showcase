# Staff Features

Library staff use the application to perform day-to-day borrowing and return operations.

## Main Staff Workflows

- log in to the system
- search students
- search books
- create rental transactions
- process returns
- inspect borrowing history
- view book details
- read and listen to book descriptions
- review ratings and comments

## Day-To-Day Value

- reduces manual tracking errors
- keeps stock and borrowing records synchronized
- helps staff find books quickly
- supports more informative service through descriptions and audio playback

## Notable UX Features

- searchable tables
- detail dialogs
- visual status labels
- chart summaries on the dashboard
- richer book detail with text and sound
