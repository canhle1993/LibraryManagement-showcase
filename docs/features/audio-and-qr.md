# Audio And QR Features

Two features make this project stand out from a typical academic desktop CRUD system.

## 1. Book Description Audio

From the book detail dialog, users can:

- open the book description
- click the speaker icon
- wait for the system to generate an audio narration
- listen to the description through the Vbee text-to-speech integration

This adds an accessibility and experience layer to the project.

## 2. QR-Based Warehouse Verification

The warehouse workflow supports scanning book QR codes.

- phone camera can be used through DroidCam
- QR codes are decoded through the ZXing-based flow
- scanned results can be checked against inventory records

This feature adds a more practical inventory-handling workflow to the application.
