# Project Summary

## Project Name

Library Management System

## Context

This was a semester project built by a 4-member team at Aptech. The goal was to create a desktop application that supports real library workflows instead of only basic book CRUD.

## Product Scope

The system covers:

- staff authentication and approval
- book rental and return
- student management
- book catalog and metadata management
- warehouse and inventory workflows
- location mapping inside the library
- reviews and ratings
- damage handling
- recycle bin recovery
- chart-based reporting

## Why This Project Is Portfolio-Worthy

- It combines business workflows, database design, UI handling, reporting, and third-party integrations in one product
- It shows desktop application engineering, not just web development
- It includes standout features like Vbee audio narration and QR-assisted inventory handling
